# NodeJS Restful API 서버
<br/>

## 목적
- NodeJS+Express 환경에서의 Restful API 서버 구현
- Typescript 설정 및 활용
- JWT 설정 및 활용
<br/>

## 환경
- BackEnd : NodeJS + Express Generator, JWT, Typescript
- FrontEnd : ejs(간단한 요청용 view)
<br/>

## 기능
- JWT 발급(Cookie), 검증, 갱신, 삭제
<br/>

## 진행중
- 사용자별 권한 제한
- Cookie가 아닌 별도의 Token 저장 공간
